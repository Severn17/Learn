# 数据结构和变量

## Number

### 不区分整数和浮点数,同一用Number

``` js
123;
0.123;
1.2345e;
-123;
Infinity; //无限大
```

### 四则运算

```js
1 + 2;
2 - 1;
1 * 2;
2 / 1;
10 % 3;
```

## 字符串

``` js
'abc';
"abc";
```

## 布尔值

``` js
true;
false;
2 > 1;
2 >= 3;
```

### &&(与),||(或),!(非)  运算符

``` js
true && true;
true || true;
！true;
```

### 比较符

``` js
2 > 5;
5 >= 2;

false == 0; // 自动转换数据类型再比较
false === 0; // 不会自动转换数据类型，数据类型不一致返回false，一致再比较
// 尽量不要使用==比较，始终坚持使用===比较

NaN === NaN // false NaN 与所有其他值都不相同，包括它自己

// 唯一能判断 NaN 的方法是通过 isNaN()的函数:
isNaN(NaN) // true

// 注意浮点数的相等比较:
1 / 3 === (1 - 2 / 3); // false

// 浮点数在运算过程中会产生误差，要比较浮点数是否相等，只能计算它们之差的绝对值，看是否小于某个阈值:
Math.abs(1 / 3 === (1 - 2 / 3)) < 0.00000001; // true

// null 和 undefined


```
